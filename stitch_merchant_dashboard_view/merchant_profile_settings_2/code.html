Roles & Permissions

Quick guide to user access levels.

Admin

Full access to all platform features, including team management, API keys, routing configurations, and financial reports.

Editor

Can modify transactions, update routing rules, and view analytics. Cannot manage team members or access sensitive security settings.

Viewer

Read-only access to transactions and analytics. Useful for support staff or external auditors who don't need configuration rights.
